Start of the git session

  1. Open Terminal
  2. Navigate to the ProjectDirectory:
    cd path/to/ProjectDirectory
  3. Initialize a Git Repository:
    git init

Git is a command-line tool, so the first thing to do is to open the terminal, navigate to the project Directory, and initialize a Git repository

Adding and committing to changes

  1. git add file
    or
    git add .
  2. git commit -m "text message"

Pushing changes

  1. git push origin main

This command will push the changed to the remote repository name (often "origin) and the branch name you want to push (main)
This command will not end the initialized session, after pushing you can continue to work on the project, make more changes, commit them and push those changes.

Creating a new branch

  1. git branch branch_name

    This command will create e new branch.

  2. git checkout branch_name

    This command will change your current branch to the branch_name branch.

  3. git merge branch_name

    This commat will take the changes committed in branch_name and merge them to the branch you are currently on
